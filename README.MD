## Motivation APP

\
\
This app was created from a study in kotlin. Its purpose is to generate random motivational phrases.


### Language:

*   Kotlin 1.8.0 
*   Java 11.0.+

<br/>


### Adroid API:
*    PIXEL 3 - API 27

    The choice of API 27 was based on the number of devices available to run the application.
